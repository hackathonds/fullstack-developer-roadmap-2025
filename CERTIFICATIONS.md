# Certification Guide & Study Plans

Recommended certifications and study plans:

1) AZ-900 — Azure Fundamentals
- 4-week study guide: cloud basics, core services, pricing & SLA, hands-on labs.
- Resources: Microsoft Learn AZ-900 path, Azure free account labs.

2) AZ-204 — Azure Developer Associate
- 8-week hands-on plan: App Services, Functions, Storage, Security, Monitoring, Service Bus.
- Resources: Microsoft Learn AZ-204 path, sample projects in this repo.

3) Meta Front-End Developer (Coursera)
- Use during Phase 2–3; complete capstone in Phase 3.

4) MongoDB Developer / DP-900 (Azure Data Fundamentals)
- Use for Phase 5 learning validation.

How to track exam readiness:
- Create issues for each knowledge area.
- Implement small labs and mark issue Done when you can deploy a working example on Azure.
