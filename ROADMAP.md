# Roadmap — 8-Phase, 28-Week Program (Summary)

This file summarizes the 8 phases with week ranges and primary outcomes.

Phase 1 — Fundamentals & Setup (Weeks 1–2)
- Outcomes: Advanced C# features, Git workflows, JS ES2023 review, dev environment ready.
- Deliverables: Console app (async), Git practice contributions.

Phase 2 — Frontend Foundations (Weeks 3–6)
- Outcomes: HTML5 semantics, accessibility, advanced CSS, DOM & APIs.
- Deliverables: Responsive portfolio, weather app.

Phase 3 — React Ecosystem (Weeks 7–12)
- Outcomes: React + TypeScript, hooks, state management, testing.
- Deliverables: Task manager (React+TS), e-commerce catalog.

Phase 4 — Backend APIs (Weeks 13–18)
- Outcomes: ASP.NET Core 8+, EF Core, JWT/OAuth, API testing & docs.
- Deliverables: Blog API, auth service, file upload service.

Phase 5 — Databases (Weeks 19–22)
- Outcomes: SQL Server/Postgres skills, MongoDB basics, performance tuning.
- Deliverables: Multi-database app, analytics queries.

Phase 6 — Cloud & DevOps (Weeks 23–28)
- Outcomes: Azure app deployment, Docker, CI/CD (GitHub Actions), monitoring.
- Deliverables: Dockerized full stack app, Azure-hosted e-commerce.

Phase 7 — Advanced Concepts (ongoing)
- Outcomes: Microservices, gRPC, GraphQL, SSR/SSG, performance & security hardening.
- Deliverables: Microservices prototype, GraphQL gateway.

Phase 8 — Portfolio & Production-readiness
- Outcomes: Production deployment, security, observability, scalability, hiring prep.
- Deliverables: Production-ready portfolio, interview prep checklist, sample architecture docs.

Use this file as the master schedule. Each phase folder contains week-by-week tasks, checklists and starter code.
